# LLM Brand Lab

当有人问AI助手"有什么好用的筋膜枪"或者"哪款项目管理工具值得推荐"时，答案是怎么来的？

我们很容易假设AI推荐是对评论、排名和品牌声誉的某种客观综合。但还有一种更令人不安的可能性：品牌的**内容写法**本身——与产品质量无关——会影响AI最终推荐哪个品牌。

如果这是真的，它的含义相当深远。它意味着AI推荐是可以被塑造的。它意味着理解这一点的品牌，会对不理解的品牌形成系统性优势。它也意味着我们正在进入一个新时代——针对AI系统的内容优化，而不只是针对搜索引擎，已经成为一门真实存在、却几乎无人系统研究的学科。

我们开始跑实验来弄清楚这件事。

---

## 我们发现了什么

实验设计很简单：取同一个产品的两段描述，事实相同，写法不同。问LLM会推荐哪个品牌。重复多次，每次交替A/B的顺序以控制位置偏差。排除回复不明确的结果，统计胜率。

在覆盖5个品类（蓝牙耳机、筋膜枪、移动电源、项目管理工具、精品咖啡）的46次实验中，DeepSeek每一次都选择了叙事风格。**46次，全部。100%。**

我们还做了分解实验——分别测试类比和开放性问题单独使用时的效果——发现两种技巧同时使用时效果最强。任何一种单独使用，结果都不一样。

## 这可能意味着什么

我们的工作假设：语言模型在海量人类写作上训练，在这些文本中，叙事结构和隐喻是质量、专业性和可信度的一贯信号。功能性要点内容，相反，与广告文案的模式高度相似——模型可能已经学会对此保持怀疑，或者至少缺乏热情。

我们观察到最有效的两种技巧：

**跨领域类比** — 将产品与来自完全不同领域的事物联系起来。风暴后重建的珊瑚礁。爵士即兴演奏。森林树冠过滤光线。领域跨度越大，读者（或模型）能够进入的概念框架就越丰富。

**开放邀请** — 不以行动号召结尾，而是以一个问题结尾，将推销变成对话。"你上一次给予恢复与训练同等重视，是什么时候？"模型像人类读者一样，被引导着去完成这个思考。

我们将这种模式称为 **AIO**——AI内容优化——类比于SEO。假设是：就像搜索引擎曾经奖励某些结构性信号（外链、关键词、页面权重），AI推荐系统可能也在奖励某些**语义信号**。区别在于，AI的信号很大程度上是未知的，几乎没有人在系统性地研究它们。

这只是假设。这正是这个项目存在的原因。

---

## 为什么我们需要更多数据

一个模型，一个研究者，46次实验。这还不足以得出结论——只够提出问题。

GPT-4o是否表现出同样的偏好？Claude呢？开源模型呢？这个效果是否在不同品类中都成立，还是只在我们测试的几个品类中特有？把prompt换成中文会改变结果吗？模型规模有影响吗？

我们不知道。找出答案需要在一系列模型上跑同样的实验，而这正是这个项目作为集体项目要完成的事。

## 运行实验

```bash
git clone https://github.com/philwong2015-svg/llm-brand-lab.git
cd llm-brand-lab
pip install openai anthropic google-generativeai  # 按需安装
```

```bash
python experiment.py --provider openai    --api-key sk-...
python experiment.py --provider anthropic --api-key sk-ant-...
python experiment.py --provider google    --api-key AIza...
python experiment.py --provider deepseek  --api-key sk-...
python experiment.py --provider openclaw  # 如果你有OpenClaw
```

大约10分钟，大多数服务商费用不到$0.10。**你的API key始终留在本地** — 脚本将结果保存为本地JSON文件，这是你唯一需要提交的东西。

## 待探索的问题

- 这个效果因模型系列而异，还是普遍存在？
- 对于B2B产品和消费品，效果强弱是否不同？
- 如果功能型描述也写得很好——不是要点列表，而是清晰有力的散文——结果会怎样？
- 当AI能够访问关于品牌的外部信息，而不仅仅依赖提供的文本时，效果是否仍然存在？
- 最关键的：这个效果能迁移到真实世界的AI推荐场景，还是只在强制A/B对比中成立？

最后一个问题才是真正重要的。其他的都是铺垫。
